# Register as a Worker on Walnut

This guide will walk you through the process of registering your device as a worker on the Walnut network.

<Callout type="info" emoji="💡">
  Before you begin, ensure your device meets the minimum system requirements and you have a stable internet connection.
</Callout>

## Registration Process

<Steps>
### Connect New Worker

Navigate to the Worker Workspace and click **Connect New Worker** to begin the registration process.

![Connect New Worker](/assets/docs/v1.0.1/provider/2.1.png)

### Enter Worker Name

Choose a unique identifier for your worker device. This name will help you track your device's performance and earnings.

<Callout type="warning">
  Worker names must be unique across the network. Choose a name that helps you identify your device easily.
</Callout>

![Enter Worker Name](/assets/docs/v1.0.1/provider/2.2.png)

### Select Operating System

Specify your device's operating system to ensure compatibility with our worker software.

![Select Operating System](/assets/docs/v1.0.1/provider/2.3.png)

### Specify Device Type

Indicate whether your device will contribute CPU or GPU resources to the network.

<Callout type="info">
  GPU workers typically earn higher rewards due to their enhanced processing capabilities for specific tasks.
</Callout>

![Specify Device Type](/assets/docs/v1.0.1/provider/2.4.png)

### Run Setup Script

Execute the provided setup script to configure your device for the Walnut network.

<Callout type="warning" emoji="⚠️">
  Make sure to review the script contents before execution for security purposes.
</Callout>

![Run Setup Script](/assets/docs/v1.0.1/provider/2.5.png)

### Authorize Device

Complete the device authorization process to ensure secure communication with the network.

![Authorize Device](/assets/docs/v1.0.1/provider/2.6.png)

### Review Deployment Result

After completing the setup, you'll see a detailed overview of your deployment:

#### Cluster Configuration Result
The system will display a comprehensive summary of your deployment, including:
- Node configurations
- Network settings
- Resource allocations
- Deployment status

![Cluster Configuration](/assets/docs/v1.0.1/provider/2.7.png)

#### Worker Node Status
Monitor your worker node's status in real-time through our dashboard. Currently, this is demonstrated through AWS simulation.

<Callout type="warning" emoji="⚠️">
  This is a simulation environment. In production, Walnut will provide executable configurations tailored to your specific environment, enabling seamless integration with your existing infrastructure.
</Callout>

![Worker Nodes](/assets/docs/v1.0.1/provider/2.8.png)

### Next Steps
After successful registration:
- Monitor your worker's performance
- Track earnings and rewards
- Adjust resource allocation as needed
- Join the Walnut provider community
</Steps>
